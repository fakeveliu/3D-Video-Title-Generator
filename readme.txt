3D Video Title Generator

Demo Video Link: https://www.youtube.com/watch?v=_HKwtrwD1u4

This program allows the user to create and save a customized 3D title intro for their videos. Simply enter a title, an image, and add effects to it to make a cool intro!

How to run: Run 3d-video-title-generator.py

Libraries: numpy & pillow

Shortcut commands: My program is quite straightforward...the only shortcut is after you enter your title, hit 'r' to get to the effects mode directly.
